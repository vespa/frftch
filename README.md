
# Docs

## Dependencies
* NodejS 8+;

### Run the client

On application's root: 

```js 
npm start
```

Run tests and coverage:
```js 
`npm test`
```
Run ESLint (avoid do that on vscode or similar, prefer a non-IDE terminal. Installed plugins can conflict)
```js 
`npm run lint`
```

To see the Documentation (up on http://localhost:6060/)
```js 
npm run docs
```
